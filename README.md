# AI-CHATBOT-INTEGRATION

## Overview

ASP.NET Core MVC application integrated with Azure Bot Service.

## Features

- AI Chat Interface
- Azure Bot Service Integration
- Interactive Responses
- Web-based Chat UI

## Technologies

- ASP.NET Core MVC
- Azure Bot Service
- Bot Framework SDK
- .NET 8

## Run

```bash
dotnet restore
dotnet run
```

Open:

https://localhost:5001
```
